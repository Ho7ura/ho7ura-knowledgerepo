  整体架构一句话

  这是一个 AI 盲人导航眼镜的服务器端：眼镜上的 ESP32 把视频/音频/IMU 数据通过 WiFi 发给电脑，电脑跑这些 Python 代码做 AI
   推理，然后把语音指令返回给眼镜。

  两个大型二进制文件

  - mobileclip_blt.ts（约 600MB）— MobileCLIP 模型权重，供开放词汇检测用
  - hand_landmarker.task（约 8MB）— MediaPipe 手部关键点检测模型（根目录和 model/ 各有一份）

  ---
  总结：全是 Python 脚本，python app_main.py 就跑起来了。需要 NVIDIA GPU 做推理，依赖阿里云 DashScope
  API。唯一需要编译器的是 compile/ 里给 ESP32 的固件。
